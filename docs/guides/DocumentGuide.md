# 문서 작성 가이드

## 폴더 구조

```
docs/
├── design/     # 게임 디자인 문서
└── guides/     # 개발 가이드 문서
```

## 문서 작성 규칙

### 문서 길이 관리
- 문서가 **100줄을 넘어가면 분리를 고려**
- 분리 기준: 주제별, 시스템별로 독립적인 문서 생성

### 두괄식 문서 작성
- **문서 상단에 핵심 내용을 먼저 요약**
- 결론이나 핵심 개념을 먼저 제시하고, 세부 설명은 그 뒤에
- 독자가 문서를 처음 열었을 때 즉시 핵심을 파악할 수 있도록

### 시스템 기획 문서 구조
시스템 기획 문서는 다음 세 가지로 분리하여 작성:
1. **시스템 요소** - 시스템을 구성하는 핵심 요소 정의
2. **동작** - 요소들이 어떻게 상호작용하는지 설명
3. **예시 데이터** - 실제 수치/밸런스 예시
