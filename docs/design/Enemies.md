# 적 유닛

기지를 향해 진격하는 적 유닛의 요소, 동작, 목록을 정의하는 문서

> 공통 요소/동작은 [유닛 공통](UnitBase.md) 참조

---

## 적 전용 요소

- **이동 속도**: 레인을 따라 이동하는 속도
- **골드 보상**: 처치 시 획득하는 골드

---

## 적 전용 동작

### 이동
- 스폰 후 레인을 따라 오른쪽→왼쪽으로 이동
- 각 유닛별 고유 이동속도 적용
- 공격 거리 내 타깃이 있다면 이동 정지, 공격 시작

### 근거리 공격
- 공격 시 즉시 대상에게 피해 적용

### 원거리 공격 [TODO]
- 공격 시 발사체 생성
- 발사체가 대상에게 도달하면 피해 적용

### 사망
- 사망 시 골드 드랍

## 유닛 목록

- **기본 적**
  - 체력: 낮음 / 공격력: 낮음 / 이동속도: 보통
  - 일반 근접 공격
  - 가장 기본적인 적 유닛

- **돌진 적**
  - 체력: 낮음 / 공격력: 낮음 / 이동속도: 빠름
  - 빠른 이동
  - 빠르게 전선을 돌파하는 유닛

- **탱커 적**
  - 체력: 높음 / 공격력: 보통 / 이동속도: 느림
  - 높은 체력
  - 전선을 오래 버티는 유닛
