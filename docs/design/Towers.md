# 아군 유닛 (타워)

플레이어가 배치하는 아군 유닛의 요소, 동작, 목록을 정의하는 문서

> 공통 요소/동작은 [유닛 공통](UnitBase.md) 참조

---

## 타워 전용 요소

- **배치 비용**: 배치에 필요한 에너지
- **쿨다운**: 배치 후 재배치까지 대기 시간

---

## 타워 전용 동작

### 파괴
- 파괴 시 해당 그리드 칸 비워짐

---

## 유닛 목록

- **기본 사수**
  - 체력: 낮음 / 공격력: 보통 / 비용: 낮음
  - 단일 타겟 원거리 공격
  - 가장 기본적인 공격 유닛

- **방벽**
  - 체력: 높음 / 공격력: 없음 / 비용: 매우 낮음
  - 공격 불가
  - 적의 진행을 막는 방어 유닛

- **에너지 생성기**
  - 체력: 보통 / 공격력: 없음 / 비용: 낮음
  - 주기적으로 에너지 생산
  - 자원 수급용 유닛
